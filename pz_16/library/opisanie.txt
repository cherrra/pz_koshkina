Main - класс, содержащий метод main для запуска всех действий. 
Создает объекты книг, библиотеки, пользователей, авторов, издательств, жанров, библиотекарей и осуществляет действия выдачи/возврата книг, бронирования книг, поиска книг

Классы:
1. Book - класс, представляющий книгу в библиотеке. 
Содержит поля: название, автор, год издания, количество страниц, количество доступных копий. 
Методы: конструктор, геттеры и сеттеры для полей, методы для уменьшения/увеличения количества доступных экземпляров.

2. Library - класс, представляющий библиотеку. 
Содержит поля: название, адрес, список книг. 
Методы: конструктор, геттеры и сеттеры для полей, методы для добавления/удаления книг из списка, метод для поиска книги по названию.

3. User - класс, представляющий пользователя библиотеки. 
Содержит поля: имя, возраст, список взятых книг. 
Методы: конструктор, геттеры и сеттеры для полей, методы для добавления/удаления книг из списка взятых книг.

4. Author - класс, представляющий автора книги. 
Содержит поля: имя, год рождения, страна происхождения. 
Методы: конструктор, геттеры и сеттеры для полей.

5. Publisher - класс, представляющий издательство книги. 
Содержит поля: название, год основания, страна расположения. 
Методы: конструктор, геттеры и сеттеры для полей.

6. Genre - класс, представляющий жанр книги. 
Содержит поля: название, описание. 
Методы: конструктор, геттеры и сеттеры для полей.

7. Librarian - класс, представляющий библиотекаря. 
Содержит поля: имя, возраст, должность. 
Методы: конструктор, геттеры и сеттеры для полей, методы для выдачи/возврата книг.

8. Loan - класс, представляющий выдачу книги пользователю. 
Содержит поля: книга, пользователь, дата выдачи, дата возврата. 
Методы: конструктор, геттеры и сеттеры для полей.

9. Reservation - класс, представляющий бронирование книги пользователем. 
Содержит поля: книга, пользователь, дата бронирования. 
Методы: конструктор, геттеры и сеттеры для полей.

10. Illustrator - класс, представляющий иллюстратора книги. 
Содержит поля: имя, опыт, страна рождения. 
Методы: конструктор, геттеры и сеттеры для полей.

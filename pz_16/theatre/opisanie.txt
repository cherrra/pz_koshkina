Main - класс, содержащий метод main для запуска всех действий. 
Создает объекты актеров, пьес, театра, директора, сцены, реквизита, костюмов, билетов и зрителей. 
Затем эти объекты используются для создания объекта Performance, который представляет собой конкретное представление 
+ вывод информации обо всех объектах в консоль.

Классы:
1. Actor - класс, представляющий актеров.
Содержит поля: имя, возраст, пол, опыт работы.
Методы: конструктор, геттеры и сеттеры для полей.

2. Play - класс, представляющий пьесу.
Содержит поля: название, автор, год написания, список актеров.
Методы: конструктор, геттеры и сеттеры для полей, метод addActor для добавления актера в список.

3. Theatre - класс, представляющий театр.
Содержит поля: название, адрес, список пьес.
Методы: конструктор, геттеры и сеттеры для полей, метод addPlay для добавления пьесы в список.

4. Director - класс, представляющий директора.
Содержит поля: имя, возраст, пол, опыт работы.
Методы: конструктор, геттеры и сеттеры для полей.

5. Stage - класс, представляющий сцену.
Содержит поля: название, размеры, список реквизита.
Методы: конструктор, геттеры и сеттеры для полей, метод addProp для добавления реквизита в список.

6. Prop - класс, представляющий реквизит.
Содержит поля: название, описание.
Методы: конструктор, геттеры и сеттеры для полей.

7. Costume - класс, представляющий костюмы.
Содержит поля: название, описание. 
Методы: конструктор, геттеры и сеттеры для полей.

8. Ticket - класс, представляющий билеты.
Содержит поля: номер, цена, дата.
Методы: конструктор, геттеры и сеттеры для полей.

9. Audience - класс, представляющий зрителей.
Содержит поля: имя, возраст, пол, место на зале.
Методы: конструктор, геттеры и сеттеры для полей.

10. Performance - класс, представляющий спектакль.
Содержит поля: пьеса, режиссер, сцена, список актеров, список реквизита, список костюмов, список билетов и список зрителей. 
Методы: конструктор, геттеры и сеттеры для полей, метод addActor для добавления актера в список, метод addProp для добавления реквизита в список, 
метод addCostume для добавления костюма в список, метод addTicket для добавления билета в список, метод addAudience для добавления зрителя в список.
